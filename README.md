# 🌐 Personal Portfolio Website

## 📌 Project Overview

This is my first personal portfolio website created as part of my journey as an Information Technology student.

The goal of this project is to create a place where I can showcase my skills, projects, and growth as I continue learning Web Development, Software Engineering, and Cybersecurity.


---

## 🎯 Problem Statement

Many students learn different technologies but do not have a platform where they can organize and present their skills and projects.

This project provides a personal space to document my learning journey and demonstrate practical skills.


---

## 💡 Solution

A responsive personal portfolio website that displays:

- Personal introduction
- Technical skills
- Projects
- Learning progress
- Contact information
